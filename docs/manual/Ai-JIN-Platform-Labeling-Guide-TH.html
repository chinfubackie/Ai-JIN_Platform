3. ใช้ Polygon และ SAM ร่วมกับกรอบ

Polygon

  1. กด Polygon (P)
  2. คลิกตามแนวขอบอย่างน้อย 3 จุด
  3. ปิดรูปและตรวจ class
  4. ลบแล้ววาดใหม่หากแนวขอบผิด

SAM2 / SAM v1

  1. เลือก SAM model
  2. กด SAM Auto (S)
  3. คลิกกลางชิ้นงานหรือใช้ rough box
  4. ตรวจ Preview แล้วกดใช้งาน

4. Auto-label, Ask LM และการบันทึก

คำสั่งการใช้งานสิ่งที่ต้องตรวจ
Auto-labelใช้โมเดลกับภาพปัจจุบันกรอบซ้ำ, class, false positive/negative
Auto-label ทั้งโฟลเดอร์ประมวลผลหลายภาพสุ่มตรวจทุก workpiece และสภาพแสง
เล่นอัตโนมัติdetect, รอ, บันทึก, ไปภาพถัดไปตั้งเวลารอให้ตรวจทัน
Ask LMเสนอวัตถุที่ขาดหรือปรับกรอบLM Status และผลก่อนยืนยัน

บันทึก

เขียน label ของภาพปัจจุบัน แล้วคงอยู่หน้าเดิมเพื่อ QA ซ้ำ

บันทึก & ถัดไป

เขียน label แล้วเปิดภาพถัดไป เหมาะกับ workflow ต่อเนื่อง

รูปแบบที่จัดเก็บ

labels/<split>/ชื่อเดียวกับภาพ.txt

Box: class cx cy width height   Polygon: class x1 y1 x2 y2 ...

อย่าเปลี่ยนภาพจนกว่าจะยืนยันว่า Save สำเร็จและตัวนับ labeled เปลี่ยน

5. Checklist คุณภาพต่อภาพ

ต้องผ่าน

  • ครบทุกชิ้นงาน
  • class ถูกต้อง
  • กรอบชิดวัตถุ
  • ไม่มีกรอบออกนอกภาพ
  • ไม่มี duplicate annotation
  • Save แล้วเปิดใหม่ยังเห็นผล

ต้องแก้

  • กรอบกินพื้นหลังมาก
  • ตัดส่วนสำคัญของชิ้นงาน
  • label เงาหรือแสงสะท้อน
  • ชื่อคลาสไม่ตรง data.yaml
  • ทำงานใน split_quarantine
